<!-- Aggergation -->
A MongoDB aggregation query processes data documents through a multi-stage data transformation pipeline to return computed, summarized results. It uses the db.collection.aggregate() method, which accepts an array of specialized stages where the output of one stage serves as the input to the next
Aggregation operations process multiple documents and return computed results. You can use aggregation operations to:

Group values from multiple documents.

Compute a single result from the grouped data.
Analyze data changes over time.
Query the most up-to-date version of your data.
